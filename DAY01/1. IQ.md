#  30 Essential Linux Commands for DevOps Engineers

Below is the improved, clean version.

---

## 1. **ls**

**Description:** List directory contents

**Syntax:**

```bash
ls [options]
```

**Example:**

```bash
ls -l
```

**Use case:** View files with permissions and ownership.

---

## 2. **cd**

**Description:** Change directory

**Syntax:**

```bash
cd <directory>
```

**Example:**

```bash
cd /etc
```

**Use case:** Navigate directories during troubleshooting.

---

## 3. **pwd**

**Description:** Print working directory

**Syntax:**

```bash
pwd
```

**Example:**

```bash
pwd
```

**Use case:** Confirm the current directory in scripts or SSH sessions.

---

## 4. **mkdir**

**Description:** Create a new directory

**Syntax:**

```bash
mkdir <directory>
```

**Example:**

```bash
mkdir logs
```

**Use case:** Create folder structures for deployments.

---

## 5. **rm**

**Description:** Remove files or directories

**Syntax:**

```bash
rm [options] <path>
```

**Example:**

```bash
rm -rf /tmp/cache
```

**Use case:** Clean temporary files or artifacts.

---

## 6. **cp**

**Description:** Copy files or directories

**Syntax:**

```bash
cp <source> <destination>
```

**Example:**

```bash
cp config.yaml /backup/
```

**Use case:** Backup config files.

---

## 7. **mv**

**Description:** Move or rename files

**Syntax:**

```bash
mv <source> <destination>
```

**Example:**

```bash
mv app.log app.log.old
```

**Use case:** Rename logs or config files.

---

## 8. **cat**

**Description:** Display file contents

**Syntax:**

```bash
cat <file>
```

**Example:**

```bash
cat /etc/os-release
```

**Use case:** Quick file content view.

---

## 9. **tail**

**Description:** View end of file

**Syntax:**

```bash
tail -f <file>
```

**Example:**

```bash
tail -f /var/log/syslog
```

**Use case:** Real-time log monitoring.

---

## 10. **head**

**Description:** View beginning of file

**Syntax:**

```bash
head <file>
```

**Example:**

```bash
head /etc/passwd
```
